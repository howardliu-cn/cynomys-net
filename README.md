## 描述

本项目源自 [cynomys](https://github.com/howardliu-cn/cynomys) 中的网络传输模块，沿用原来的模块名称，整体进行重构。希望在这个过程中，打造一个更加通用的RPC基础库，并借助该项目重新整理Netty4相关知识。
